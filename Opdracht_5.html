Knop Historie_deel2

De opdracht
Maak 4 knoppen naast elkaar.

Op de ene knop staat A, de andere B die daarnaast C en die daarnaast D.

Op het moment dat je op knop C van de knoppen klikt verschijnt er in een div:
U heeft op knop C geklikt.

Op het moment dat je op knop D klikt verschijnt er in een div.
U heeft op knop C geklikt.
U heeft op knop D geklikt.

Hij houd dus jouw knop-klik history bij.

Maar als je hierna op C klikt;
Verschijnt er een alert: U heeft al een keer op C geklikt.
en verschijnt er in een div.

U heeft op knop C geklikt.
U heeft op knop D geklikt.
U heeft op knop C geklikt.

ADVANCED

De alert vertelt je hoe vaak je al op C had geklikt. LET OP: Als je de browser refreshed is alles weer leeg.
